== Problème avec le build

[source,yaml]
....
image: node:4.2.2   

all_tests:
  script:           
   - npm install express --save
   - node ./myapp.js
....

Non :

- il demande l'installation d'un outil extérieur
- il ne fait pas de test

Pas bête :

- "Je ne vois pas de problème"

== _Delivrables_ attendus

Vous finaliserez avant le *vendredi 17/01/2020* votre dépôt gitlab (pensez à ajouter votre enseignant de TP et `jmbruel` comme contributeur) :

- Un rapport (le `README` de votre dépôt) contenant (outre les éléments habituels d'un rapport
comme les noms et contact des binômes, une table des matières, ...)
une courte explication par chaque fonctionnalité nouvelle ou refactoring précis
avec des extraits de code illustratifs et une justification.
