Attendus du projet

Modèles à réaliser

On ne vous embête pas ce coup-ci avec les modèles mais n’hésitez pas à en utiliser (des cohérents avec votre code) pour vos documentations.

Delivrables attendus

Vous finaliserez avant le vendredi 17/01/2020 votre dépôt gitlab (pensez à ajouter votre enseignant de TP et jmbruel comme contributeur) :

  • Un rapport (le README de votre dépôt) contenant (outre les éléments habituels d’un rapport comme les noms et contact des binômes, une table des matières, …​) une courte explication par chaque fonctionnalité nouvelle ou refactoring précis avec des extraits de code illustratifs et une justification.

Delivrables attendus

  • Les codes Java: à minima un répertoire src avec les sources et un répertoire doc avec la javadoc.

Evaluation et critères

  1. démonstration que ça fonctionne pareil
  2. pertinence des choix
  3. qualité du rapport (illustration, explications)
  4. qualité du code
  5. extras (tests, modèles)
